# What is Environmental Racism?
An explorable explanation about environnmental racism and the historical and policy mechanisms that lead to disproportionate exposure of marginalized groups to environmental hazards. Designed and built by Felicia Yan for EDUC 432: Designing Explorable Explanations for Learning in Spring 2026.
